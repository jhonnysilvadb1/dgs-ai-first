# Trilha de Formação para Certificação AI First da DGS
 
Este repositório reúne os materiais da **Trilha de Formação para Certificação AI First da DGS**.
 
## Objetivo
 
Preparar os papéis de engenharia da DGS — DMs, Profissionais de Produtos (BA e UX), Lead Engineers, Devs e QAs — para o novo **SDLC AI First da DGS**.
 
## Tópicos abordados
 
- Fundamentos de IA Generativa
- Engenharia de Prompt
- Engenharia de Contexto
- RAG e MCP
- AI Agents
- Recorte de Domínio e Spec Driven Development
- Agents.md
- Skills
- Harness Engineering
- Revisão Crítica de Outputs de IA
